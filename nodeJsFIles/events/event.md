# Event Module

È um mecanismo para
* [ ] Disparar eventos
* [ ] Ouvir eventos
* [ ] Fazer alguma ação quando ouvir o evento
* [ ] É a base para muito outros módulos como: https, stream, file system, etc..


## O que aprendi

* [ ] Utilizar eventos
* [ ] Dispara eventos
* [ ] Ouvir eventos
* [ ] Executar ações para determinados eventos
* [ ] Entender como ele é a base para os outros módulos
